Overview

This Bedrock Client communicates with the server for voice data.

Features

Captures audio on Bedrock platform

Sends encoded packets to server

Plays incoming audio streams

Setup

Install the Bedrock add-on.

Ensure permissions for microphone.

Launch the world/server.

Go to Minecraft Bedrock → Settings → Creator:

Enable these:

✔ Script Engine
✔ Enable GameTest Framework
✔ Beta APIs
✔ Device permissions
✔ Device.Audio API

If these settings do NOT appear:
→ You must use Minecraft Preview (Bedrock Beta) from Xbox Insider Hub.

Development

Located in the Bedrock client add-on pack

Uses Bedrock scripting API for networking where available
